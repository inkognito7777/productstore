# ProductStore API

Backend project for managing products with metadata and Telegram bot integration.

## Tech Stack
- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Maven
- PostgreSQL

## Features
- CRUD operations for products
- Product metadata support
- Telegram bot integration
- Layered architecture (Controller / Service / Repository)

## Architecture
- REST API design
- DTO-based communication
- Global exception handling

## Future improvements
- JWT authentication
- Swagger documentation
- Docker support
- Unit tests
